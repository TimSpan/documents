                                        # 深入理解javascript原型和闭包

                                        # （1）——切都是对象

                                        对象——若干属性的集合。

                                        java或者C#中的对象都是new一个class出来的，而且里面有字段、属性、方法，规定的非常严格。但是javascript就比较随意了——数组是对象，函数是对象，对象还是对象。对象里面的一切都是属性，只有属性，没有方法。那么这样方法如何表示呢？——方法也是一种属性。因为它的属性表示为 键值对的形式。javascript中的对象可以任意的扩展属性，没有class的约束。
